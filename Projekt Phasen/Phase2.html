#13
API Dokumentation mit Swagger einrichten
FastAPI generiert automatisch eine interaktive Dokumentation unter /docs. Jede Route bekommt eine kurze Beschreibung — was sie macht, was sie erwartet und was sie zurueckgibt. Das Team kann dann alle Endpunkte direkt im Browser testen ohne extra Tools.

So sieht eine dokumentierte Route aus:
@app.post("/auth/register",
    summary="Neuen User registrieren",
    description="Erstellt einen Account mit E-Mail und Passwort.")
def register(user: UserCreate):
    ...
