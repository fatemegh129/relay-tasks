## 🚀 Was ist Relay?

Wir bauen eine App für unser Team 🛠️

Die Idee ist einfach — stell dir vor du hast eine Aufgabe die aus mehreren Schritten besteht.
Jeder im Team übernimmt seinen Schritt, macht ihn fertig und gibt die Aufgabe weiter 🏃‍♂️➡️🏃‍♀️

Auf dem Dashboard sieht man sofort wer gerade was macht,
wer auf jemand anderen wartet und wo etwas feststeckt 👀
